# 图书馆智能服务系统

## 项目概述

本项目旨在构建一个深入理解用户隐性需求的图书馆智能服务系统，超越传统管理系统仅满足表面需求的局限，真正触达用户的深层需求。

## 核心理念

从"查书找座位"的表面需求，深入挖掘：
- 考研学生的长期稳定复习空间需求
- 留学生的英文文献导航需求
- 教师的新书追踪需求
- 焦虑学生的安静角落需求

## 开发阶段

### 第一阶段：场景发掘与需求定义
- 实地观察记录
- 用户访谈
- 需求层次模型构建
- 优先级评估

### 第二阶段：架构决策与智能体协作开发
- 架构决策记录(ADR)
- 核心系统开发
- AI协作过程记录

### 第三阶段：验证审计与反思迭代
- 真实用户试用
- AI偏见审计
- 工程反思文档

## 技术栈

- Python 3.10+
- 推荐算法：协同过滤 + 内容匹配 + 情境感知混合
- 数据库：SQLite (开发) / PostgreSQL (生产)
- Web框架：FastAPI

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行系统
python src/main.py
```
